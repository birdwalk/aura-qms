# AURA QMS — Competition Demo Script

## Setup (30 seconds)
1. Open the app — the AI Command Center loads immediately
2. Point out: "No login screen. Judges see value on the first pixel."
3. Show the "Viewing as" dropdown — explain the multi-user architecture

## Demo Flow (5 minutes)

### Step 1 — Management View (90 seconds)
- Dashboard is set to "QMS Manager" view by default
- Walk through the 5 metric cards: readiness score (count-up animation), findings, docs at risk
- Point to the agent activity feed: "Six agents are continuously monitoring compliance"
- Show the two critical compliance alerts — Operations and Procurement
- Click the readiness trend chart: "Improving from 68% to 76% over 6 months"

### Step 2 — Ask AURA Chat (90 seconds)
- Click the suggested prompt: "Which departments need urgent attention?"
- Show the typing indicator → agent name appears in the response header
- Point to sources citation at the bottom: "Grounded in knowledge base — not hallucinated"
- Type: "Generate a corrective action plan for Operations"
- Show the multi-step response: agent decomposes task, cites findings, gives prioritised steps

### Step 3 — Process Owner View (60 seconds)
- Switch "Viewing as" to "Operations Manager (PO-004)"
- Dashboard changes to Personal Portal
- Show: "Same system, different lens — this is what the process owner sees"
- Show their personal learning path with the overloaded workload warning
- Show their own documents and findings — scoped only to their department

### Step 4 — Manager Insights (30 seconds)
- Switch back to QMS Manager
- Navigate to Manager Insights
- Show the bar chart, donut chart, and workload table
- Read the AI-generated executive briefing: "This is what goes to the board"

### Step 5 — Compliance Index (30 seconds)
- Navigate to Compliance Index
- Show ISO 9001:2015 clause-by-clause status
- Highlight red clauses: "The agents mapped every finding to its clause — automatically"

## Closing Statement
"AURA QMS is not a chatbot. It is six specialised agents orchestrated by Microsoft Foundry,
grounded in ISO knowledge, connected to organisational signals through Work IQ, and reasoned
through the Fabric IQ semantic layer. Every response is cited. Every action is traceable.
Every insight is built for humans to act on — not just to read."
